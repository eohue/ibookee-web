# 🛠️ Render 영구 디스크 설정 가이드 (Final Action)

이 가이드는 이미지가 사라지는 문제를 해결하기 위해, Render 서버에 **영구 디스크(Persistent Disk)**를 부착하는 구체적인 절차입니다.
코드 수정은 필요 없습니다. Render 웹사이트에서 설정만 하시면 됩니다.

---

## ⚠️ 준비 사항
*   이 기능은 **유료 플랜(Starter, 월 $7)**부터 사용할 수 있습니다.
*   아직 무료(Free) 플랜이라면 먼저 업그레이드가 필요합니다.

---

## 1단계: 디스크 추가하기

1.  [Render 대시보드](https://dashboard.render.com/)에 접속합니다.
2.  목록에서 **`ibookee-web`** (Service Name)이라는 글자를 클릭해서 상세 페이지로 들어갑니다.
    *   *주의: `ibookee Renewal 2026` (Project)가 아닙니다. 그 아래에 있는 `ibookee-web`을 누르셔야 합니다.*
3.  이제 왼쪽 메뉴에 **[Disks]** 탭이 보일 것입니다. 클릭하세요.
4.  만약 안 보인다면 **[Settings]** 탭으로 가서 **Disk** 섹션을 찾거나 유료 플랜 업그레이드를 먼저 진행해야 할 수 있습니다.
5.  **[Add Disk]** 버튼을 클릭합니다.

## 2단계: 설정값 입력하기 (정확히 입력해주세요!)

다음 내용을 복사해서 해당 칸에 붙여넣으세요.

*   **Name (이름)**:
    ```text
    ibookee-assets
    ```
    *(원하는 다른 이름으로 하셔도 됩니다)*

*   **Mount Path (마운트 경로) ⭐️중요⭐️**:
    ```text
    /opt/render/project/src/attached_assets
    ```
    *(이 경로가 틀리면 이미지가 저장되지 않습니다. 띄어쓰기 없이 정확히 입력해주세요!)*

*   **Size (크기)**:
    ```text
    1 GB
    ```
    *(이미지만 저장하므로 1GB면 충분합니다. 나중에 부족하면 늘릴 수 있습니다)*

## 3단계: 확인 및 재배포

1.  **[Create Disk]** 버튼을 누릅니다.
2.  Render가 자동으로 서버를 재시작(Redeploy)합니다. (약 1~2분 소요)
3.  배포가 완료되면(초록색 `Live` 표시), **이제부터 업로드하는 이미지는 영원히 사라지지 않습니다!**

---

### ✅ 기존에 업로드했던 이미지는요?
이미 서버에서 사라진 파일들은 복구할 수 없습니다.
설정이 완료된 후, **이미지가 엑스박스(X)로 나오는 게시물들은 [수정] 버튼을 눌러 이미지를 다시 한번 등록**해 주셔야 합니다. 이후로는 안전하게 보관됩니다.

설정을 마치셨나요? 완료하시면 알려주세요!
